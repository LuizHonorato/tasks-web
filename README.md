# Introdução

Tasks Web - Aplicação para cadastro de tarefas.

# Pré-requisitos

- Node 18.x
- NPM ou Yarn

# Sobre o build

Esta aplicação frontend foi criada com a utilização das seguintes ferramentas:

- React
- Typescript
- React Router Dom
- Chakra UI

# Instalação

Para instalar as dependências do projeto, rode o seguinte comando:

`npm install`

ou

`yarn`

# Observação

Antes de executar o frontend, execute a aplicação backend através das instruções desse [repositório](https://github.com/LuizHonorato/tasks-server).

# Execução

Após a instalação das dependências, rode o comando:

- `npm run start`

ou

- `yarn start`
 
Após iniciar, a aplicação estará disponível no endereço: http://localhost:3000.
